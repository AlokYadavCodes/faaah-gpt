# FaaahGPT

Makes ChatGPT say “Faaah”.

It listens to ChatGPT responses and plays the sound when a matching word appears.  
You can add your own trigger words.

## Install

- [Chrome Web Store](https://chromewebstore.google.com/detail/dinkemaklnlgbjjdepnipehfkhdapjfl?utm_source=item-share-cb)  
- [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/faaahgpt-faaah-for-chatgpt/)
